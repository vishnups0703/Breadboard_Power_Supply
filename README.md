# Breadboard Power Supply

![PCB 3D Render or Physical Photo](docs/images/3d-render.png)

A compact, plug-and-play power supply module designed to mount directly onto a standard MB102 breadboard(160mm x 55mm x 10mm). It regulates an external DC input down to clean, dual-rail voltages to power prototyping circuits safely.

## 🚀 Key Features & Specs
* **Output Voltages:** Dual-rail 3.3V and 5V DC, selectable via physical jumper switches.
* **Input Power:** 7V - 12V DC via a standard 2.1mm barrel jack.
* **Form Factor:** Custom PCB pitch matching the outer power rails of standard MB102 breadboards.
* **Safety Features:** Onboard power LED indicator.
## Regualtors Used
* 5V Regualator-LM7805_TO220
* 3V Regualator-LM317_TO-220
